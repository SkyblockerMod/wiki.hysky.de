Follow this step-by-step guide to **install Skyblocker** and get started.

## 📜 **Requirements**
Before installing, make sure you have:
- **Minecraft** (`latest` version)
- **Fabric**

## 📥 **Downloading the Mod**
1. **Go to the official download page**:
   - [🔗 Modrinth](https://modrinth.com/mod/skyblocker-liap/)
   - [🔗 CurseForge](https://www.curseforge.com/minecraft/mc-mods/skyblocker)

2. **Download the latest version** of the `.jar` file.

## 🛠️ **Installing the Mod**
1. **Install Fabric Loader**:
   - Download the [Fabric Installer](https://fabricmc.net/use/)
   - Run the installer and choose the correct Minecraft version.
   - Click **Install**.

2. **Place the mod file**:
   - Move `skyblocker.jar` into the `mods` folder inside `.minecraft`.

3. **Install Fabric API**:
   - [Download](https://modrinth.com/mod/fabric-api) and place `Fabric API` in the `mods` folder.

4. **Launch the game** with the **Fabric** profile.

## 🎮 **Verifying Installation**
- Install [ModMenu](https://modrinth.com/mod/modmenu/) and move into the `mods` folder inside `.minecraft`.
- Open **Minecraft** and go to the **Mod menu**.
- Check if `Skyblocker` appears in the list.

🚀 **You’re all set!** Now you can proceed to [configuration](./configuration).
