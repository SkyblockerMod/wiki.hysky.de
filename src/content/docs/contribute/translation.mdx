# Translating Skyblocker

Skyblocker uses [Weblate](https://translate.hysky.de) to manage translations.

Anyone can contribute ‚Äî no coding skills required!

---

## Getting Started

1. **Visit:** [translate.hysky.de](https://translate.hysky.de)
2. **Sign up or log in.**
3. **Select your language.**
4. Start translating strings directly on the Weblate platform!

---

## Recognition & Credits

All translators will:

- Be **credited in the README**.
- Receive a special **Translator tag** in our [![Discord](https://img.shields.io/discord/879732108745125969?logo=discord&labelColor=cecece&color=7289DA&label=%E2%80%8B)](https://discord.com/invite/aNNJHQykck) server.

---

## Guidelines

- Aim for **accurate and clear translations**.
- Keep formatting placeholders (e.g., `{0}`, `%s`) intact.
- Consistency matters! If unsure about terminology, feel free to ask in Discord.
